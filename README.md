# 个人介绍网站（GitHub Pages）

本站已按学术主页风格重制，包含 3 个页面：

- `index.html`：主页 + 近期科研进展
- `about.html`：介绍页（教育/工作/研究方向）
- `works.html`：工作成果页（论文/项目/GitHub）

配套文件：

- `styles.css`：统一样式（简洁学术风）
- `script.js`：论文、年份分布、仓库列表等动态内容

数据来源：

- `Publications.enw`
- GitHub 公开仓库信息
- ResearchGate 个人页面入口（用于主页链接补充）

## 部署方式

1. 新建仓库（推荐：`luvega.github.io`）。
2. 上传网页文件与资源文件（含 `avatar.jpg`）。
3. 仓库 `Settings -> Pages`：
   - Source：`Deploy from a branch`
   - Branch：`main` / root
4. 等待部署完成后访问：`https://luvega.github.io/`
